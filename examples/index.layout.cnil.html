Saisissez votre phrase
Pour un mot de passe de douze caractères ou plus, votre phrase doit contenir au moins :
  • Une douzaine de mots
  • Un nombre
  • Une majuscule
  • Un signe de ponctuation ou un caractère spécial (dollar, dièse, ...)

Le mot de passe

Est associé à la phrase :
  • Attention : votre mot de passe ne contient pas de minuscule
  • Attention : votre mot de passe contient peut-être une date de naissance
  • Attention : votre mot de passe contient une suite de chiffre
  • Attention : votre mot de passe contient des caractères qui se suivent dans l'ordre alphabétique
  • Attention : votre mot de passe contient des caractères qui se suivent sur le clavier
  • Attention : votre mot de passe contient un nombre trop important de caractères identique qui se suivent
Pour le retenir
  • Mémorisez exactement la phrase que vous avez choisi
  • Ne conservez que les premières lettres de chaque mot
  • Conservez les majuscules
  • Conservez la ponctuation et les caractères spéciaux
  • Conservez le(s) chiffre(s)
> Conseils pour renforcer votre mot de passe
La robustesse de votre mot de passe peut être améliorée en appliquant les modifications suivantes :
  • Eviter de choisir un nombre compris entre 1950 et 2049
  • Ajouter une ponctuation ou un caractère spécial (dollar, dièse, ...)
  • Utiliser un emoticône pour remplacer le mot correspondant, comme remplacer le mot "sourire" par :-)

  • Attention : votre mot de passe contient des caractères qui se suivent sur le clavier

Pour le retenir
  • Mémorisez exactement la phrase que vous avez choisi
  • Ne conservez que les premières lettres de chaque mot
  • Conservez les majuscules

  • Conservez le(s) chiffre(s)
  • Transformation des chiffres alphanumériques en chiffres numérique

> Transformations proposées pour renforcer votre mot de passe
  • La conversion des nombres écrits en lettres par des nombres en chiffres n'est pas utilisée
  • La conversion d'un mot par caractère spécial (ex : un par 1, clin d´oeil par ;-) ) n'est pas utilisée
